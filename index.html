About Me

Currently, I am a post-doc at Tsinghua University. I received my PhD degree from Tsinghua University in 2023, and Bachelor degree from Huazhong University of Science and Technology (HUST) in 2018. I build distributed memory/storage systems with fast and smart hardware.

Research

  • Lifting network-level knowledge to storage software
    • Leveraging NIC hardware ACKs for fast commit (NSDI'25)
  • Distributed protocols with network supports
    • In-network cache coherence (FAST'21)
    • RDMA primitives for replication (OSDI'23)
  • Indexes for different network links
    • NUMA-aware index (OSDI'21, TOS'22)
    • RDMA-based index on disaggregated memory (SIGMOD'22, EuroSys'25)
  • Memory management
    • Memory compaction (USENIX ATC'22)
    • Protective remote memory (FAST'23)
    • SSD-extended remote memory (FAST'24)
  • Concurrency
    • One-sided locks (FAST'23, FAST'25)
    • Request scheduling (USENIX ATC'22)
    • Transactions (VLDB'22, DAC'20)
